# sample_google_maps_flutter

2つのパッケージを比較するためのプロジェクトです。\
ここでは「google_maps_flutter」パッケージのサンプルアプリを公開します。

## 比較基準
- 現在地を表示できるか
- 現在地の表示アイコンを変更できるか
- 現在向いている方角を表示できるか
- 任意の場所にピンを立てることができるか
- ピンのアイコンは変更可能か
- ピンの大きさは変更可能か
- 地図そのものの見た目を柔軟に変更できるか
    - 夜の雰囲気を出せる色合いに変更可能か
    - 地形などの表示内容をデフォルメ可能か
    - 地形情報などをできるだけ削ぎ落とした白地図のような形にできるか
- 料金がどのくらいかかるか
- ピンを立てることや地図利用そのものに料金は発生するか
- パッケージのAPIは使いやすいか
